# рҹҢұ Crop Prediction System using Machine Learning

A Machine Learning based web application that predicts the most suitable crop based on environmental and soil parameters such as temperature, humidity, rainfall, pH, Nitrogen (N), Phosphorus (P), and Potassium (K).

The system is developed using Flask, Scikit-learn, HTML/CSS, and Weather API integration.

---

# рҹҡҖ Features

- Crop prediction using Machine Learning
- Weather information using WeatherAPI
- Interactive Flask web application
- Dynamic state and district selection
- Feature importance visualization
- AJAX-based weather updates
- API rate limiting using Flask-Limiter
- Environment variable protection using `.env`
- Responsive UI
